# License — choose one before you publish

This scaffold doesn't pick a license for you. For an open hardware + firmware
project, a common, clean split is:

- **Hardware** (`hardware/`): CERN-OHL-S or CC-BY-SA 4.0
- **Software** (`firmware/`, `tools/`, `sim/`): MIT or Apache-2.0
- **Docs** (`docs/`): CC-BY-4.0

Pick, drop the real license text in here (and per-folder if you split them),
then delete this note.
